# vue-pure-pdpa

## Project setup
```
npm install
npx spago install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Run your end-to-end tests
```
npm run test:e2e
```

### Run purescript tests
```
npx spago test
```

### Lints and fixes files
```
npm run lint
```
